# Aufgabe: Summe zweier Zahlen

Sie erhalten zwei ganze Zahlen als Eingabe über `stdin`, getrennt durch ein Leerzeichen.
Geben Sie die Summe der beiden Zahlen aus.

## Eingabeformat
Zwei Integer-Werte `a b` (mit optional führenden oder nachfolgenden Leerzeichen).

## Ausgabeformat
Eine einzelne Zeile mit `a + b`.

## Beispiel
Eingabe:
```
2 3
```
Ausgabe:
```
5
```

## Hinweise
- Verwenden Sie `sys.stdin.read()` oder `input()`.
- Achten Sie auf mögliche führende oder nachfolgende Leerzeichen.
- Negative Zahlen sind möglich.
